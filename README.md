# 안산 출장마사지 사이트

경기도 안산시 전지역 방문 관리 서비스(출장마사지·홈타이) 안내 정적 사이트입니다.

**상호**: 토끼24 마사지  
**예약전화**: 0508-202-4719

## 구조

- **정적 HTML 사이트** — 어느 호스팅(GitHub Pages, Netlify, 웹서버)에서든 그대로 서빙 가능
- **build.py** + **content/** — 페이지를 Python으로 정의하고 정적 HTML 생성
- **생성물** — 각 디렉터리의 `index.html`, `sitemap.xml`, `robots.txt`

```
build.py                    # 빌드 스크립트
content/
  site.py                  # 상호·전화·도메인·메뉴
  main.py                  # 메인 페이지
  areas.py                 # 지역별 페이지 (21개)
  stations.py              # 역세권 페이지 (13개)
  areas_and_stations.py    # 생활권 페이지 (12개)
  info.py                  # 정보 페이지 (예약·가이드·약관)
assets/
  style.css                # 럭셔리 다크 + 오렌지 + Pretendard
  nav.js                   # 모바일 네비게이션
gyeonggi/ansan/
  index.html               # 생성된 메인 페이지
```

## 빌드

```bash
python3 build.py
```

빌드 시 페이지별 본문 글자수 리포트가 출력됩니다.

## SEO 운영 원칙

- 본문 **2,000자 미만 페이지는 자동 `noindex`** 처리
- 지역은 21개 동 페이지
- 역은 13개 역 페이지 (환승역도 URL 하나)
- 생활권은 12개 페이지 (지역+역 조합)
- 상단·하단 메뉴에 키워드·지역명 대량 나열 없음
- 모든 페이지 본문은 고유 작성 (지역명만 바꾼 복붕 없음)

## 배포 전 할 일

1. `content/site.py`의 `BASE_URL`을 실제 도메인으로 변경
2. `python3 build.py` 재실행
3. Google Search Console에 `sitemap.xml` 제출
4. 지역별·역세권·생활권 페이지 2차 작성

## 특징

- **럭셔리 다크 팔레트**: 어두운 네이비 배경 + 오렌지 #FF6B35 액센트
- **Pretendard 폰트**: 한국식 산세리프 폰트 (최상의 가독성)
- **글래스모르피즘**: 반투명 오버레이, 블러 효과
- **반응형 디자인**: 모바일·태블릿·데스크톱 완벽 지원
- **접근성**: WAI-ARIA, 시맨틱 HTML, 키보드 네비게이션
- **성능**: 정적 HTML (CDN 친화적, 로딩 속도 빠름)

## 컨텍스트

안산시는 상록구와 단원구로 구분되며, 각 지역의 생활권을 명확히 구분하여 사용자가 예약 전 정확한 정보를 확인할 수 있도록 설계되었습니다. 구글 상위노출 정책(E-E-A-T, 도움되는 콘텐츠, Who/How/Why)을 반영하여 신뢰성 높은 콘텐츠를 제공합니다.
